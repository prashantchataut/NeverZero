# Product Overview

**Never Zero** is an Android productivity app focused on habit tracking and streak maintenance.

## Core Features

- **Streak Tracking**: Users build and maintain daily habits with visual progress tracking
- **Vocabulary Builder**: Learn new words with integrated reading sessions
- **Reading Tracker**: Track reading sessions and books
- **Daily Reflections**: Journal entries with AI-powered insights
- **Achievements**: Gamification system with badges and points
- **Time Capsules**: Future-dated messages and reflections
- **AI Integration**: Gemini-powered personalized quotes and Buddha-style insights
- **Smart Notifications**: Context-aware reminders and "ghost" notifications to maintain engagement

## User Experience Philosophy

- **Never Zero**: Core principle - maintain at least one active streak at all times
- **Skill Paths**: Organize habits into learning journeys
- **Freeze Days**: Allow users to pause streaks without breaking them
- **Personalization**: AI-driven motivational content based on user context and progress
- **Gamification**: Points, achievements, and leaderboards to drive engagement

## Key User Flows

1. Onboarding → Create first streak → Daily logging → Achievement unlocks
2. Template library → Import curated habits → Customize → Track progress
3. Reading session → Vocabulary capture → Spaced repetition learning
4. Daily reflection → AI insight generation → Time capsule creation
